# Invoice OCR Extraction (AI Powered)

Pipeline untuk extract data invoice dari PDF/Image menggunakan:
- Tesseract OCR
- OpenCV preprocessing
- Google Gemini LLM

---

## 🚀 Features
- OCR multi-language (EN, DE)
- PDF & Image support
- LLM structured extraction (JSON)
- Validation layer
- Retry + Throttling

---

## ⚙️ Setup

```bash
git clone https://github.com/Yusufalfi/pdf-extraction.git
cd pdf-extraction
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt


```
## RUN
python main.py
